# Gn !

Gn ( **G**ood **n**igh ) is a simple app that you can `clone` it into `/home/user` , compile `main.c` , copy binary file into `/bin` and run it by command .

<hr>

### Ok lets start

- Clone into ~
```commandline
cd ~
git clone https://github.com/BlackIQ/Gn
cd Gn
```

- Compile main.c
```commandline
gcc main.c
```

- Rename a.out to Gn
```commandline
mv a.out Gn
```

- Copy Gn to /bin
```commandline
sudo cp -r Gn /bin
```
- Run Gn command
```commandline
Gn
```
